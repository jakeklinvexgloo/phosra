# Phosra LinkedIn Services — Draft

These are the services to add to the Phosra LinkedIn company page. Each entry includes a short name and a 1-2 sentence description suitable for the LinkedIn Services section.

---

## 1. PCSS (Parental Controls Software Specification)

An open specification that normalizes 67 child safety laws into 45 standardized API rule categories. PCSS lets platforms adopt a single spec so parents can define protection rules once and enforce them everywhere.

## 2. Platform Safety API

A REST API that enables platforms to integrate parental controls, policy enforcement, and compliance verification. Includes API keys, rate-limited endpoints, an interactive MCP sandbox playground, and full developer documentation.

## 3. Safety Compliance Tracker

A living database tracking 67+ child safety laws across U.S. federal, state, EU, UK, Australian, and international jurisdictions. Filterable by status, jurisdiction, and rule category, with automated weekly monitoring for legislative updates.

## 4. Independent Safety Research

Comprehensive, transparent child safety evaluations across AI chatbots and streaming platforms. Published results include safety grades, standardized test prompts, scoring methodology, and head-to-head platform comparisons.

## 5. Platform Safety Directory

A directory of 320+ apps and platforms in the children's ecosystem, categorized by integration tier and safety capabilities. Includes parental control tools, streaming services, social media, gaming platforms, and AI chatbots.

## 6. Developer Tools & SDKs

API documentation, quickstart guides, a PCSS specification reference, and an interactive playground for testing policy creation, enforcement, and platform queries. Designed for platform engineers integrating child safety controls.

## 7. Platform Safety Scorecard

The first independent child safety accountability index, ranking AI chatbots and streaming platforms on protection practices. Transparent A+ through F scoring with published methodology and full test results.
